Morse code is a universal communication system. This tool lets you transmit messages visually using your phone or computer screen, even if you don't know Morse code yourself.

🛠️

How to Use

  1. Type the message you want to send.
  2. The Morse dots and dashes will appear automatically.
  3. Click Transmit Morse Flash.
  4. Hold your screen up to the recipient. The screen will blink the code!
